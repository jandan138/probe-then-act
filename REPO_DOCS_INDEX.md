# REPO_DOCS_INDEX

Put the `docs/` folder from this package directly under the repository root:

```text
probe-then-act/
  docs/
    00_foundation/
      00_PROJECT_BRIEF.md
      01_REPO_BLUEPRINT.md
    10_protocols/
      02_EXECUTION_PLAYBOOK.md
      03_EXPERIMENT_PROTOCOL.md
```

## Reading order
1. `docs/00_foundation/00_PROJECT_BRIEF.md`
2. `docs/00_foundation/01_REPO_BLUEPRINT.md`
3. `docs/10_protocols/02_EXECUTION_PLAYBOOK.md`
4. `docs/10_protocols/03_EXPERIMENT_PROTOCOL.md`

## Usage
Give these four files to the research team and require:
- one written understanding memo after reading,
- weekly milestone reports based on `docs/10_protocols/02_EXECUTION_PLAYBOOK.md`,
- and all experiments to follow `docs/10_protocols/03_EXPERIMENT_PROTOCOL.md`.
