# REPO_DOCS_INDEX

Put the `docs/` folder from this package directly under the repository root:

```text
probe-then-act/
  docs/
    00_PROJECT_BRIEF.md
    01_REPO_BLUEPRINT.md
    02_EXECUTION_PLAYBOOK.md
    03_EXPERIMENT_PROTOCOL.md
```

## Reading order
1. `docs/00_PROJECT_BRIEF.md`
2. `docs/01_REPO_BLUEPRINT.md`
3. `docs/02_EXECUTION_PLAYBOOK.md`
4. `docs/03_EXPERIMENT_PROTOCOL.md`

## Usage
Give these four files to the research team and require:
- one written understanding memo after reading,
- weekly milestone reports based on `02_EXECUTION_PLAYBOOK.md`,
- and all experiments to follow `03_EXPERIMENT_PROTOCOL.md`.
