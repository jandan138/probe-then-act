## Round 4: Real Genesis renders (systematic-debugging skill)

User pushed back on the matplotlib schematic; demanded real Genesis renders. Applied `superpowers:systematic-debugging`:

**Phase 1 — Root cause investigation.** Layer-by-layer instrumentation traced the 32x32 output to a single line in `genesis/ext/pyrender/offscreen.py`:

```python
@property
def viewport_width(self):
    """int : The width of the main viewport, in pixels."""
    return 32          # <-- hardcoded debug leftover
    # return self._viewport_width
```

**Phase 2 — Pattern.** `git blame` shows the `return 32` was committed by Genesis maintainer Zhou Xian on 2024-12-18 (commit `73f9d2d2`). The setter stores the real value to `self._viewport_width`; the getter ignores it. This is an upstream Genesis bug — anyone using OSMesa software rendering on this Genesis checkout will hit the same 32x32 ceiling, regardless of `cam.res` or backend.

**Phase 3 — Hypothesis.** Uncomment the two getters so they return `self._viewport_width` / `self._viewport_height`. Add `_viewport_width = _viewport_height = 1280x960` defaults in `OffscreenRenderer.__init__` (configurable via `GS_OFFSCREEN_INIT_RES` env var) so the OSMesa platform buffer is allocated at the right size before any camera is attached. Verified at 640x480: `cam.render()[0].shape == (480, 640, 3)` with real pixel content.

**Phase 4 — Apply.** Patched 2 files in the local Genesis fork (editable install):
- `genesis/ext/pyrender/offscreen.py`: uncomment the getters; add init defaults

Then rendered all 3 material families at 1280x960 with material-distinguishing surface colours (sand=tan, snow=white, elastoplastic=lavender). Composited into a 3-panel figure (`figures/gen_fig5_scene_schematic.py` was rewritten to read the 3 PNGs and lay them out side-by-side). The Genesis renders show real material-mechanic differences: sand spreads, snow is wider/loose, elastoplastic stays as a tight box (high Poisson's ratio).

Final paper: 9 pages, 5,739 words (+8 from round 3), 0 overfull, 0 undefined refs.
